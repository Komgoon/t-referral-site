Dette nettstedet (heretter kalt "nettstedet") tilbyr Tesla-henvisningskoder og relatert informasjon. Brukere samtykker i å bruke nettstedet under følgende betingelser.

1. Tjenestens formål

Dette nettstedet drives for å dele uoffisielle Tesla-henvisningskodeinformasjon og informere om tilhørende fordeler. Dette nettstedet har ingen direkte tilknytning til Tesla, Inc.

2. Inntekts- og partnerskapsopplysninger

Noen lenker på nettstedet er henvisningslenker (Referral Links). Hvis brukere kjøper produkter eller registrerer seg via disse lenkene, kan operatøren motta provisjon eller fordeler. Bruken av disse henvisningslenkene medfører ingen ekstra kostnader for brukeren.

3. Informasjonens nøyaktighet

Informasjonen som publiseres på nettstedet er basert på pålitelige kilder, men kan endres over tid. Vi kan ikke garantere 100% nøyaktighet og aktualitet. Brukere bør sjekke retningslinjene på Teslas offisielle nettsted på kjøpstidspunktet.

4. Ansvar for eksterne lenker

Dette nettstedet inneholder lenker til Teslas offisielle nettsted og tredjeparters nettsteder. Vi tar ikke ansvar for innholdet, sikkerheten eller personvernpraksisen til eksterne nettsteder.

5. Bruksbegrensninger og endringer

Operatøren kan endre eller avslutte deler av eller hele nettstedet uten forvarsel. Disse brukervilkårene kan også endres ved behov.
